Using as a dependency:

```xml
<repositories>
  <repository>
    <id>bananalang</id>
    <url>https://bananalang.github.io/maven</url>
  </repository>
</repositories>

<dependency>
  <groupId>io.github.bananalang</groupId>
  <artifactId>JavaBanana-Compile</artifactId>
  <version>(one of the versions below)</version>
</dependency>
```
